Para fazermos alterações em commits ja feitos, ou simplesmente voltar no passado, para isso:



Cria um novo commit e desfaz as alterações do outro:
    1.Pegar o id do commit:
    git log
    2. Basta:
    git revert (id_Commit)



Reverte o commit e dele ele:
    1. Verificar os commits: 
        git-log
    2. Depois disso:
        git reset --hard (id_Commit)

        --HEAD is now at b95bc28 Reapply "nome_Do_Commit_que_vai_ser_deletado"
    3. Agora que definimos qual vai ser o HEAD, podemos:
        git reset --hard HEAD~1 
        HEAD~1 = ultimo
        HEAD~2 = penultimo
        HEAD~3 = antepenultimo
