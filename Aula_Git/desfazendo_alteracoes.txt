Para fazermos alterações em commits ja feitos, ou simplesmente voltar no passado, para isso:

1. Verificar os commits 
    git-log
2. Pegar o id do commits
    commit 2ad48c068dc9677fb57efec70620700410f976b0
3. A partir dele você pode dar o revert 
    git revert 2ad48c068dc9677fb57efec70620700410f976b0
