## Criação de Diretórios  
- Use a pasta **home** utilizando o comando **cd**
- Crie o diretório utilizando o <mkdir> <nome_diretorio>

## Removendo Diretórios **VAZIOS**
<span style="color:red;">**️ Atenção: Arquivos apagados com `rm` são irrecuperáveis! ️**</span>
- rmdir <**nome_diretorio**>

## Removendo Diretórios **NÃO VAZIOS**
<span style="color:red;">**️ Atenção: Arquivos apagados com `rm` são irrecuperáveis! ️**</span>
- rm -ri <**./nome_diretório**>

## Renomeando Diretórios
- mv <**NOME_DIRETORIO**> <**NOVO_NOME**>

## Movendo Subdiretórios
- Use o comando dentro do diretório <mv> <caminho_diretorio> <caminho_destino>
**Dica** : pegue o caminho usando o pwd
