# Top, o gerenciador de tarefas

- **PID** = numero do processo

- **PR** = prioridade dele (mais alto = prioridade)

- **NI** = prioridade (19 < -20)

- **VIRT** = memória virtual 

- **S** estado (s = sleeping, t = stopped, r = running)




## Ordenando por cpu e mem
- **c** por cpu
- **m** por memória
